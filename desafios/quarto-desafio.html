Tags de Mídia

Tag img

Fotos em jpg ficam com o fundo branco, imagens em png ficam sem fundo(transparência). Já imagens em svg são vetoriais, não perdem a resolução. Quando utilizar imagem svg utilizar a tag svg.

O atributo title na tag img é importante para quando o usuário passar o mouse sobre a imagem e aparecer uma mensagem

O atributo alt serve para que o leitor de tela leia aquela imagem para usuários que tenham algum tipo de deficiência visual.


Tag audio

Atributo controls serve para colocar as opções de pause, tempo de áudio, volume...

Já o atributo autoplay serve para quando o usuario abrir o site começar a tocar o áudio


Tag video

Caso o navegador não tenha suporte a vídeos mp4, ele tentará execurtar o webm (web movie) e caso não suporte também ele executará a mensagem.

Para colocar legenda utiliza-se a tag track com um arquivo em vtt.

Tag iframe
